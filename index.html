Testing IDE tools

发表于 2020-07-23 更新于 2020-07-24 分类于 testing

关于几款测试开发工具的比较

这是一个很久以前整理的内容了，当初研究自动化测试开发，为找到最好的测试工具，分别尝试使用了一下。

IDE pycharm(intellibot\robot support) robotframework-ride RED(nokia) vscode
图形编辑界面 无 表格填写 切卡+表格填写 无
图形关键字提示助手快捷键 无 ctrl space（alt space） alt / 无
文本编辑关键字提示助手快捷键 自感应 ctrl space（alt space） alt / 自感应
断点调试 不支持 不支持 支持 需要插件
明显bug 轻微 某些关键字需要单引号才能识别，但拿到ride或red后报错 有 某些情况下，提示无法保存，权限xxx;执行界面中文乱码； 加入新的库 需要编译，偶有报错 sikuli 无
执行测试 需插件或配置扩展工具 直接执行 直接执行 需要配置
查看库python源码 完美 不支持 无法ctrl+鼠标左键直接找到 不支持
查看keyword文档 ctrl q ctrl m ctrl + 鼠标左键（或鼠标悬停） 不支持
控制单个用例执行 不支持 支持（复选框） 支持（鼠标右键） 不支持
编写自定义python指令 完美支持 不支持（需在其他ide编写后导入） 安装pydev插件后，支持python开发 支持
查询keyword 无 F5 无，但是有列表 无
